Learning Loss for Test-Time Augmentation

发表于2023-01-02|Paper Notes

Introduction

训练了一种loss prediction network来预测测试图像在对应transformation下的loss从而选择适当的transformation方法作为test-time augmentation。
提出的loss prediction network是紧凑的，减小了计算消耗。此外它能减少所需的test-time transformations，进一步减少了计算消耗。

Method

  • 传统test-time augmentation
    xxx是输入图像, τ\tauτ是一个transformation operation. 如果我们选择 T={τ1,τ2,⋯ ,τ∣τ∣}\Tau=\{\tau_{1},\tau_{2},\cdots,\tau_{|\tau|}\}T={τ1​,τ2​,⋯,τ∣τ∣​}作为参选的augmentation集，传统的test-time augmentation可以被公式化为

ytta=1∣T∣∑i=1∣T∣Θtarget(τi(x))y_{tta}=\frac{1}{|\Tau|}\sum^{|\Tau|}_{i=1}\Theta_{target}(\tau_{i}(x)) ytta​=∣T∣1​i=1∑∣T∣​Θtarget​(τi​(x))

其中 $ \Theta_{target}$ 是目标数据集上训练的神经网络

  • 本文提出的方法
    使用本文提出的loss predictor可以从 Tk={τ1,τ2,⋯ ,τk}⊆T\Tau_{k}=\{\tau_{1},\tau_{2},\cdots,\tau_{k}\}\subseteq \TauTk​={τ1​,τ2​,⋯,τk​}⊆T 中选择一个optimal set Tk⋆\Tau_{k}^{\star}Tk⋆​ 那么test-time augmentation可以被表示为

ytta⋆=1∣Tk⋆∣∑i=1∣Tk⋆∣Θtarget(τi⋆(x))y_{tta}^{\star}=\frac{1}{|\Tau_{k}^{\star}|}\sum_{i=1}^{|\Tau_{k}^{\star}|}\Theta_{target}(\tau_{i}^{\star}(x)) ytta⋆​=∣Tk⋆​∣1​i=1∑∣Tk⋆​∣​Θtarget​(τi⋆​(x))

和传统方法相比，本文提出的方法能减少测试时的计算消耗，且若选择Tk⋆\Tau_{k}^{\star}Tk⋆​的时间能被忽略，则所选出的optimal set Tk\Tau_{k}Tk​越小，测试时消耗就越小。为了实现这一步，本文提出了loss predictor Θloss\Theta_{loss}Θloss​

  • Loss Prediction Module: Architecture
    本文用EfficientNet-B0作为Θloss\Theta_{loss}Θloss​

  • Loss Prediction Module: Train Method
    使用ranking loss surrogates来优化Spearman correlation
